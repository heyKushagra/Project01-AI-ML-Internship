# Netflix Movies & TV Shows - Exploratory Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset from Kaggle. The objective is to understand the structure of Netflix's content library, identify trends and patterns, create visualizations, and derive meaningful business insights from the data.

This project was completed as part of the Pluto Academy AI & Machine Learning Internship Program.

## Dataset

**Dataset Name:** Netflix Movies and TV Shows

**Source:** https://www.kaggle.com/datasets/shivamb/netflix-shows

The dataset contains information about Netflix titles, including:

* Type (Movie/TV Show)
* Title
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre

## Project Objectives

* Load and inspect the dataset
* Clean and preprocess the data
* Perform exploratory data analysis
* Create visualizations to identify trends and patterns
* Generate business insights based on findings

## Technologies Used

## Data Cleaning

The following preprocessing steps were performed:

* Handled missing values in categorical columns
* Removed records with critical missing information
* Converted date columns to datetime format
* Verified duplicate records
* Prepared data for analysis and visualization

## Exploratory Data Analysis

The analysis focused on answering the following questions:

1. What is the distribution of Movies and TV Shows on Netflix?
2. Which countries contribute the most content?
3. How has Netflix content addition changed over the years?
4. What are the most common content ratings?
5. Which genres are most frequently available on Netflix?

## Visualizations

The project includes:

* Bar Chart
* Horizontal Bar Chart
* Line Chart
* Pie Chart
* Histogram
* Heatmap

## Key Insights

* Movies represent a larger share of Netflix's catalog than TV Shows.
* Netflix experienced significant content growth during the late 2010s.
* A small number of countries contribute the majority of available content.
* Mature audience ratings dominate the platform.
* Drama, Comedy, and International content are among the most popular genres.
